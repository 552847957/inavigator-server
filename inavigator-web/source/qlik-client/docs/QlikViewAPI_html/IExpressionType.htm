IExpressionType


Expression display options object (trendlines etc.)


Property Detail

DisplayEquation (Boolean, read-write)
True if trend line equations to be displayed.


DisplayR2 (Boolean, read-write)
True if trend line R2 coefficient to be displayed.


Order (Byte, read-write)
Order of polynomial trendline.

Order of polynomial trendline
(only used for IExpressionType.Type=1)
0 = mean
1 = linear
2 = 2nd degree fit
3 = 3rd degree fit
4 = 4th degree fit


Type (Byte, read-write)
Type of expression result to show.

Type of trendline
1 = polynomial
2 = exponential
