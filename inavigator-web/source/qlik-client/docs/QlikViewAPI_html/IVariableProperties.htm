IVariableProperties


Properties object for QlikView Variable


Property Detail

Comment (String, read-write)


ConstraintsFlag (Byte, read-write)
Type of input constraint to be used.

Type of constraint used
0 = No constraint
1 = Standard constraint
2 = Custom constraint
3 = Predefined values only
4 = Read-only


dFrom (Double, read-write)
First value of predefined number series.


dStep (Double, read-write)
Last value of predefined number series.


dTo (Double, read-write)
Increment value of predefined number series.


EnableExpressionEditDlg (Boolean, read-write)
True if full Edit Expression dialog should be available for variable in input box.


ErrorMessage (IStringExpr, objectref.)
Custom defined error message to be shown on illegal entry.


IconType (Byte, read-write)
Type of icon to be used for variable in input boxes.

Type of icon to be used for variable in input boxes
0 = No icon used
1 = Drop-down
2 = Scroll control


IncludeInBookmark (Boolean, read-write)
True if variable to be included in bookmarks.


InputConstraint (String, read-write)
Custom input constraint expression.


MRUCount (Byte, read-write)
Number of recent values in MRU list.


Name (String, read-write)
Name of variable.


NumberPresentation (IFieldAttributes, objectref.)
Number format for variable.

Number format descriptor object


OnChangeActionItems (IArrayOfActionItemDef, objectref.)
Actions triggered when a variable is changed.

Action Items triggered by variable change


OnInputActionItems (IArrayOfActionItemDef, objectref.)
Actions triggered when input is set to a variable.

Action Items triggered by variable input


PreDefinedList (Variant (Array of String), read-write)
Array of predefined values (not including number series).


StandardType (Byte, read-write)
Standard constraint.

Standard constraint
0 = Number
1 = Positive number
2 = Integer
3 = Positive integer
4 = Single character


UseMRU (Boolean, read-write)
True if recent values should be shown in dropdown in input box.


UseMsgOnError (Boolean, read-write)
True if error message to be shown on illegal entry.


UsePredefListedValues (Boolean, read-write)
True if predefined value list to be used.


UsePredefNumberSeries (Boolean, read-write)
True if predefined number series to be used.


UseScroll (Boolean, read-write)
True if predefined values to be used with scroll control in input box.


UseSoundOnError (Boolean, read-write)
True if sound should be used on illegal input.
