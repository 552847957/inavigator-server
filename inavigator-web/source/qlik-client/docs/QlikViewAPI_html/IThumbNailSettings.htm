IThumbNailSettings


Document thumbnail settings


Property Detail

HorizontalAdjust (Byte, read-write)
Horizontal alignment.

0=Right
1=Left
2=Center


ImagePath (String, read-write)
File path of document thumbnail.


ImgStretch (short, read-write)
Stretch mode of document thumbnail.

0=No stretch
1=Fill
2=Keep aspect
3=Fill aspect
4=Tile
5=Use complete


Type (short, read-write)
Type of document thumbnail.

0=None
1=Opening sheet
2=Thumbnail image


VerticalAdjust (Byte, read-write)
Vertical alignment.

0=Top
1=Bottom
2=Center
