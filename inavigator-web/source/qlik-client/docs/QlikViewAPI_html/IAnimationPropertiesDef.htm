IAnimationPropertiesDef

Property Detail

AutoPlay (Boolean, read-write)
True if animation to play automatically on opening and after selection.

True if first dimension is to be used for animation.

True if animation to return to first frame after full play.

Font for chart background display of animation dimension values.

Font definition object

Number of frames per second in animation.

Loop (Boolean, read-write)
True if animation to loop indefintely.


ShowAnimationDimension (Boolean, read-write)
True for chart background display of animation dimension values.


ValueTimeInterval (IValueExpr, objectref.)
Number of milliseconds between animation dimension values.


VerticalAdjust (Byte, read-write)
Vertical alignment of chart background display of animation dimension values.

Vertical alignment of chart background display of animation dimension values
