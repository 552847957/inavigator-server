IColorAreaDef


Color area definition object


Property Detail

Alpha (Byte, read-write)
Sets the transparency of a color area.

Transparency (0-255)


FillDirection (short, read-write)
Gradient direction.

Gradient direction
0 = horizontal
1 = vertical
2 = diagonal down
3 = diagonal up
4 = from corner
5 = from center


FillPattern (short, read-write)
Gradient pattern.

Gradient pattern
0 = upper left dialog option
1 = upper right dialog option
2 = lower left dialog option
3 = lower right dialog option


Luminosity (Byte, read-write)
Luminosity for complementary color in one-color gradients.

Luminosity for complementary color in one-color gradients
0 is black
255 is white


Mode (short, read-write)
Color area mode.

Color area mode
0 = solid color
1 = one color gradient
2 = two color gradient


PrimaryCol (IColorDef, objectref.)
Primary color.


SecondaryCol (IColorDef, objectref.)
Secondary color for two-color gradients.
