ICellImageStyle


Cell image style definition


Property Detail

ImgFallBack (Byte, read-write)
Image fallback.

Fallback if no image available
0 = blank cell
1 = show text


StretchMode (short, read-write)
Stretch mode.

Stretch mode
0 = No Stretch
1 = Fill
2 = Keep Aspect
3 = Fill with Aspect
4 = Tile
