IArrayOfBorderStylesDef


Array of border styles for cell (borders between cells in text based sheet objects) used when rendering a cell on screen. Border styles are calculated based upon the object, its current layout and cell variations. The border styles cannot be changed by the user. In order to change the appearane of a cell, the CellVariation must be used.


Property Detail

Count (Integer, read-only)
Returns number of elements in array.


Item (IBorderStylesDef, objectref.)
Returns border styles for a cell.
