IBorderStyleDef


General border style description object for one specific cell border (borders between cells in text based sheet objects) used when rendering a cell on screen. Border styles are calculated based upon the object, its current layout and cell variations. The border styles cannot be changed by the user. In order to change the appearane of a cell, the CellVariation must be used.


Property Detail

Color (IColorDef, objectref.)
Border color.

Border color as ARGB


ExtraBorderSpace16 (Integer, read-write)
Extra spacing after cell in 16th of line height.


LineStyle (Byte, read-write)
Line style for border.

Line style for border
0 = none
32 = single
64 = double
96 = two single lines with space inbetween
127 = triple
