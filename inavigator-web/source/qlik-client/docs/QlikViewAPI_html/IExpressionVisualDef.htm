IExpressionVisualDef


Chart expression visual formatting descriptor object


Property Detail

Contains ShowAsBar setting for combo when in bar/line mode and the same for line/bar when in combo mode. This property is automatically switched with IExpressionVisual.ShowAsBar when going between line/bar and combo and vice versa


BckAsLine (Boolean, read-write)
Contains ShowAsLine setting for combo when in bar/line mode and the same for line/bar when in combo mode. This property is automatically switched with IExpressionVisual.ShowAsLine when going between line/bar and combo and vice versa
