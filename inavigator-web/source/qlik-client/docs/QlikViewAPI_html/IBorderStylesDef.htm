IBorderStylesDef


Border style description object for all four cell borders (borders between cells in text based sheet objects) used when rendering a cell on screen. Border styles are calculated based upon the object, its current layout and cell variations. The border styles cannot be changed by the user. In order to change the appearane of a cell, the CellVariation must be used.
