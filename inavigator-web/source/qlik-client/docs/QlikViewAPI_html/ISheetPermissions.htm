ISheetPermissions


User permissions for a specific sheet


Property Detail

AccessSheetObjectProperties (Boolean, read-write)
True if access to sheet object properties permitted in sheet else false.


AccessSheetProperties (Boolean, read-write)
True if access to sheet properties permitted in sheet else false.


AddSheetObject (Boolean, read-write)
True if adding new sheet objects permitted in sheet else false.


CopyCloneSheetObject (Boolean, read-write)
True if copying/cloning of sheet objects permitted in sheet else false.


MoveSizeSheetObject (Boolean, read-write)
True if moving and sizing of sheet objects permitted in sheet else false.


RemoveSheet (Boolean, read-write)
True if removal of sheet permitted else false.


RemoveSheetObject (Boolean, read-write)
True if removal of sheet objects permitted in sheet else false.
