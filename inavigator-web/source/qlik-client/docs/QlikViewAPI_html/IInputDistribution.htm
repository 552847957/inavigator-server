IInputDistribution


Input field distribution definition object


Property Detail

AffectMode (short, read-write)
Input field compensation mode.

Compensation mode
0 = No compensation
1 = Compensation over selected values
2 = Compensation over all values


DistributionMode (short, read-write)
Input field distribution mode.

Distribution mode
0 = Equal parts of the change is distributed to all the underlying values
1 = The change is distributed proportionally (to existing values) to the underlying values
2 = The value entered is given to all the underlying values
3 = The value is divided equally between underlying values, without regard to previous distribution but keeping previous sum


MemberType (Integer, read-write)
Type of distribution used.

0 = same distribution as with InputSum function
3 = same distribution as with IInputAverage function
